Funções presentes nas Expressões Regulares


    Funções Java

    1. java.util.regex.Pattern:
      A classe Pattern é responsável pela representação do padrão a ser compilado e utilizado nas pesquisas. Para a sua utilização, deve-se obter uma referência de Pattern por meio do método estático compile (String padrao).

      Ex: Pattern pattern = Pattern.compile(“teste”);

    2. java.util.regex.Matcher:
      A classe Matcher é responsável pelas operações de pesquisa e análise de cadeias de caracteres de entrada, possibilitando a execução de buscas, retorno de resultados, extração de índices de informações e dados agrupados. Para a sua utilização, deve-se obter uma referência de Matcher por meio do método matcher(String texto) de uma referência de Pattern.

      Ex: Matcher matcher = pattern.matcher(“Executando um teste de RegEx…”);

    3. java.util.regex.PatternSyntaxException:
      A classe PatternSyntaxException permite o controle e tratamento de exceções lançadas durante o uso de expressões regulares. Através da verificação desse tipo de exceção, é possível ter um controle maior sobre os erros gerados a partir de consultas quando a sintaxe do padrão informada à aplicação é inválida.


    Funções PHP (POSIX)

    1. ereg_replace: Substituição com expressões regulares.

    2. ereg: Casando expressões regulares.

    3. eregi_replace: Substitução com expressões regulares insensíveis a maiúsculas e minúsculas.

    4. eregi: Aplica expressões regulares insensíveis a maiúsculas e minúsculas.

    5. split: Separa strings em array utilizando expressões regulares.

    6. spliti: Separa strings em array utilizando expressões regulares insensíveis a maiúsculas e minúsculas.

    7. sql_regcase: Cria expressões regulares insensíveis a maiúsculas e minúsculas.


    Funções PHP (PCRE)

    1. preg_filter: Executar uma pesquisa de expressão regular e substituir.

    2. preg_grep: Retorna as entradas do array que combinaram com o padrão.

    3. preg_last_error: Retorna o código de erro da última regex PCRE executada.

    4. preg_match_all: Executar uma correspondência de expressão regular global.

    5. preg_match: Executar uma correspondência de expressão regular.

    6. preg_quote: Adiciona escape em caracteres da expressão regular.

    7. preg_replace_callback_array: Executar uma pesquisa de expressão regular e substituir usando retornos de chamada.

    8. preg_replace_callback: Executa uma busca usando expressão regular e modifica usando um callback.

    9. preg_replace: Realiza uma pesquisa por uma expressão regular e a substitui.

    10. preg_split: Divide a string por uma expressão regular.

O que é Expressão Regular?

Expressão Regular é uma linguagem de busca de padrões, onde dizemos que o padrão (pattern) do texto que queremos encontrar, passamos o texto alvo (target) e pedimos para uma Regex engine (motor) fazer essa busca.
